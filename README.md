# Nutri Manager

Sistema de gestão de consultório para nutricionistas, com agenda, pacientes e diário alimentar.

## Tecnologias

- Backend: Node.js, Express, Prisma, MySQL  
- Frontend: HTML, CSS, JavaScript puro  
- Autenticação: JWT  
- Deploy local: Railway, Vercel ou similar
